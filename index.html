  • Desenvolvi uma aplicação CRUD (Create, Read, Update, Delete) completa em Google Apps Script para controle de estoque de produtos, utilizando Google Sheets como banco de dados.
  • O projeto foi entregue como um app web, proporcionando à cliente um controle de estoque na palma da mão, com funcionalidades para verificar quantidades, status e auxiliar nas decisões de compra.
  • Implementei funcionalidades de gerenciamento de inventário, incluindo: listagem visual com status por cor (Em estoque, Comprar novamente, Esgotado), valor total do estoque em tempo real, pesquisa dinâmica e filtros por status.

  • Estruturei o código em módulos (`GlobalConfig`, `WebApp`, `DataHandler`, `CrudOperations`) para melhor organização e manutenção do backend.
